# 💳 Credit Card Fraud Risk Analysis Dashboard

## 📌 Overview
The **Credit Card Fraud Risk Analysis Dashboard** provides a comprehensive view of fraudulent transactions, risk levels, transaction patterns, and regional distribution of fraud cases.  
It is designed to help organizations **detect fraud trends, minimize financial losses, and strengthen risk management strategies**.

---

## 🎯 Key Objectives
- Monitor fraud rate and transaction risk levels  
- Identify high-risk fraud types and categories  
- Analyze monthly fraud trends and patterns  
- Detect high-risk regions contributing to fraud  
- Support data-driven fraud prevention strategies  

---

## 📊 Key Metrics (KPIs)
| Metric | Value | Description |
|------|------|-------------|
| Fraud Rate | **28.6%** | Percentage of transactions identified as fraudulent |
| Total Fraud Cases | **286** | Total number of fraud incidents |
| Fraud Exposure Amount | **$3.31M** | Total financial loss due to fraud |
| Critical Risk Transactions | **10.7%** | Percentage of high-risk transactions |
| Top Fraud Type | **Card Not Present** | Most common fraud category |
